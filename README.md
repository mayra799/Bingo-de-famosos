# 🎬 Bingo de Famosos

Um jogo interativo de bingo com celebridades brasileiras e internacionais! Marque os famosos para completar linhas, colunas e diagonais.

## 🎮 Como Jogar

1. **Clique nas celebridades** para marcá-las no cartão
2. **Complete linhas, colunas ou diagonais** para fazer BINGO
3. Use o botão **"Novo Cartão"** para gerar um novo jogo
4. Use **"Resetar Marcações"** para limpar suas marcas

## 🌟 Categorias de Celebridades

- **⚽ Jogadores de Futebol**: Neymar, Pelé, Ronaldinho, Messi, Maradona e mais
- **🎤 Cantores**: Anitta, Ivete Sangalo, The Weeknd, Beyoncé e mais
- **🎭 Personagens**: Homem-Aranha, Batman, Superman, Capitão América e mais
